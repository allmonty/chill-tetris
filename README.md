# Chill Tetris

A relaxing, gesture-controlled Tetris game for mobile, built with Flutter and
the [Flame](https://flame-engine.org) engine — with a swappable color palette
and fully code-synthesized sound and music.

See **[PROJECT.md](PROJECT.md)** for full project context: architecture,
mechanics, theming, level configuration, the audio system, build/run/test
instructions, and project history.
